## TASK-016: AI plan persistence in task workflow
**Priority:** P1 | **Tags:** core, feature
**Updated:** 2026-03-20 00:00

When AI moves a task to In Progress (or starts implementing a plan), the plan should be saved in IN_PROGRESS.md before implementation begins. When the task is done, a short summary of the plan should be preserved in DONE.md after the description. The updated agent instructions (CLAUDE.md, .cursorrules) generated by the "Initialize AI Instructions" command should include these conventions.
